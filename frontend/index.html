How to make a Minecraft Mod

Before You Start



Welcome to Knodemy's tutorial on how to make a Minecraft mod! This tutorial is made to run on a 13" screen or larger.



To the right you will see the Knodemy Java code editor. This is where you will be writing your code.


1 - Creating a Constructor




Okay, lets get started learning to mod!

At the bottom of the existing file, you can see the blank Block class denoted by the line: public class TrampolineBlock extends Block We need to create a constructor for the block, so that Minecraft knows what to do when we spawn this block. A constructor describes what to do every time our block is created in the game. In the constructor we will describe the basic details of the trampoline block.

We can do this by creating a constructor below the line denoted by the comment.
public TrampolineBlock(){}


Type
public TrampolineBlock(){}

2 - Defining a Constructor




Now in between the curly braces of the constructor, we will describe the basic behavior of our Trampoline block. First, we’ll inherit the qualities of an existing block. There is no need to make a completely new set of rules, so we will use the rules from another block, specifically sand. We’re using sand, because it’s easy to break with just your hands in Minecraft. To do this we must call the constructor of the generic block to use the sand material.
super(Material.sand);

1. Add all of these lines to your constructor:
super(Material.sand);

2 - Defining a Constructor





Next we have to add our block to the GameRegistry. The GameRegistry organizes all of the blocks in the game and makes sure that each block has a different name. To register the game we will call the registerBlock function.
GameRegistry.registerBlock(this, name);

You probably noticed that there is two different arguments inside the parentheses: this and name. this tells the GameRegistry that we are registering the block we are currently constructing, the trampoline. If you look above the constructor, you can see we declared a variable with the name TrampolineBlock.name gives the GameRegistry the name we want to use for the block.

Add these lines to your constructor.
GameRegistry.registerBlock(this, name);

3 - Completing the Constructor




Also, we need to set the name of the block for the other parts of the game. To do that we just need this one line:

setUnlocalizedName(name);


Finally we will tell Minecraft where our block belongs in creative mod. You’ve probably noticed that there are different tabs in your creative inventory that make it easy to find the item or block you want. Since our trampoline is a block, we will put it in the block category.

setCreativeTab(CreativeTabs.tabBlock);

Add these lines to your constructor. setUnlocalizedName(name);
setCreativeTab(CreativeTabs.tabBlock);

4 - Connecting the Block and Mod




Now we're going to work on the main mod class.
If you scroll up, you can see the mod class indicated by the line public class TrampolineMod

Our block has been defined, but Minecraft does not know our block exists! As a result, we need to create a mod class to get the game’s attention. The TrampolineMod class’s purpose is to tell Minecraft about all the items and blocks in our mod and how to access those things. As you can see, there are already two functions in the file: preinit and init.

Before the Minecraft world loads up, we need to tell the game what our mod includes. That is what the the preinit function does; it tells Minecraft what blocks and items we are adding and puts them in the GameRegistry. In our preinit, we only need one line, because we only have one block. This line will create an instance of our block.

trampolineBlock = new TrampolineBlock();


Add this line to the preinit() function.
trampolineBlock = new TrampolineBlock();

5 - Recipe Components




Next we have the init function. This is where the mod is set-up and where we will describe the recipe to create the trampoline block. First we need to define what will be created with this recipe. Obvioiusly we want to create the trampoline block.

ItemStack i = new ItemStack(trampolineBlock);

1. Create an ItemStack
ItemStack i = new ItemStack(trampolineBlock);

6 - Creating the Recipe




Next, we need to describe the recipe. Those of you who have played Minecraft before know that the crafting table provides a 3 by 3 grid of squares to place ingredients in. We will define our recipe in as if it were a 3 by 3 grid. This recipe also goes in the init function.
Object recipe = new Object[]{
“PPP”,
“PWP”,
“PPP”,
};

Notice I split up the recipe onto different lines, so it is easier to recognize as a 3 by 3 grid. Instead of writing out the name of each item, we are using the letters P and W to stand for wooden planks and wool, respectively. So our recipe stands for a block of wool, surrounded by wooden planks. Now that you understand that, we must tell Minecraft what our letters mean. These two elements go in the Object array after the last comma.
'P', Blocks.planks, ‘W’, Blocks.wool

1. Your recipe Object Array should look like this:
Object recipe = new Object[]{
“PPP”,
“PWP”,
“PPP”,
'P', Blocks.planks, 'W', Blocks.wool
};

7 - Finalizing the Recipe




Finally, we need to associate the block and the recipe together and register the connection. Like before where we registered to block with the GameRegistry, we need to register our recipe with the GameRegistry using the ItemStack and Object array we just created.This line still goes in the init function.

GameRegistry.addRecipe(i, recipe);


That’s it! Our recipe is done!
1.Register the recipe:
GameRegistry.addRecipe(i, recipe);

8 - Bounciness




We have defined our block, added it to a mod, and created the recipe. Now to finish it off, we need to make it bouncy. On a trampoline, you can only bounce if you jump on it. So we need to call a function that will make the character bounce if you jump on it. The function we need is called onFallenUpon.

Return your focus to the block class below. We can create the function below the respective comment with this line of code:

public void onFallenUpon(World world, BlockPos bp, Entity entity, float f){}


There are a lot of arguments there, but we only are worried about the entity. entity refers to any living thing that jumps on our trampoline block.

1.Add the function onFallenUpon:
public void onFallenUpon(World world, BlockPos bp, Entity entity, float f){}

9 - Bounciness Continued!




To move the player, we will tell the entity to move using its moveEntity function. So between the brackets of the function you created in the last step. Add this line.

entity.moveEntity(entity.serverPosX, entity.serverPosY+5, entity.serverPosZ);


As you can see, we refer to the entity many times in that line. we don’t want to move the player horizontally, so we do not change its server position on the x or z axes. The y-position or its vertical position is what we want to change, so we add five to its current position.

1.Call the moveEntity function from that function:
entity.moveEntity(entity.serverPosX, entity.serverPosY+5, entity.serverPosZ);

You're Done!



Congratulations on finishing your first Minecraft Mod!
Now we will go through how to install your mod into Minecraft.

Installing Forge




First, you'll need to install Minecraft Forge. Minecraft Forge is a community made tool used to easily create and install mods and helps them work together.
First head over to Forge's Download page and download the recommended installer.

Once you download the installer. Run the installer using Java by selecting 'Open With' Java.
Download the installer from http://files.minecraftforge.net/
Run the installer using java.

Installing the Mod




Next we need to install our mod.
If you click on the "Submit Code" button, we will process your code and give you the mod file as a .jar.

Windows
1. Go to the start menu, type "run", and press enter.
2. In the window that pops up type "%AppData%".
3. The file explorer will popup and you need to go to the folder named "minecraft".

Mac
1. Open Finder.
2. Hold down ALT and click "Go" then "Library" in the top menu bar.
3. Open the folder "Application Support" and then for the "minecraft" folder.

Both
4. Next you need to open the "mods" folder.
5. Finally you need to drag the jar file you created into this mod folder.

Congratulations!
