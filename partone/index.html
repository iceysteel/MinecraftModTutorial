5 - While Loops




While loops loop a portion of the code as long as a condition is true They take the format

}

  1. Lines of code before it to initiate some variables (not necessarily needed)
  2. The construction of the while loop
  3. The conditional
  4. And the code looped

6 - For loops




For loops loop a portion of the code a certain number of times They take the format
for(int i = 0; i<5; i++){
      System.out.println(i);
}
As you can see, the for loop has 3 things inside the parentheses
  1. Initiated variable
  2. Conditional
  3. counter
The purpose of the counter is so that you don't forget to add it at the end of the looped code. The rest of the loop has the same format as the while loop

The above for loop does the exact same thing as the task with while loops

7 - If/else if/else




Conditional statements exectute if a condition is true/not true

if(x < 4){
     x++;
}
elif(x==4){
     x=0;
}
else{
     x--;
}

If/else if/else statements are checked in the order they are written and only execute once
  1. If the condition in the if statement is true, execute code in the if statement
  2. If the condition in the else if statement is true and ALL ABOVE conditions are false, execute code in that else if statement
  3. Repeat step 2 for however many else if statements
  4. If all conditions in ALL if and else if statements are false, execute code in the else statement
Make an if/else if/else statement inside a for loop so that when the counter satisfies a condition, break the loop.

8 - Object Oriented Design




Java is an Object Oriented programming language


Every variable you have used so far is an object, and generally, each object has its own class.

There are 3 things associated with objects
  • Constructors
  • Methods
  • Fields
This is just an introduction to making functions in classes. Move on after you understand that Java programming is made of objects.

9 - Class structure




Classes are organized into fields, constructors, and methods

public class Sword {
    int durability, damage;
    public Sword(int a, int b){
        durability = a;
        damage = b;
    }
    public void removeDurability(){
        durability--;
    }
}

As you can see, fields are initiated without values outside of any function or constructor. Usually they get values in the constructor. The constuctor is usually underneath the fields. Methods change something in the object and are underneath the constructor. Note the brackets and no public static void main(String[] args)

This is also just more information for you.

11 - The Constructor




The constructor initiates the object and assigns values

public class Sword {
    int durability, damage;
    public Sword(int a, int b){
        durability = a;
        damage = b;
    }
}

The constructor takes in values and uses them in the code in the construction, so when you type in Sword sword = new Sword(999,999); then you will create a sword with 999 durability and 999 damage.

12 - Methods




The constructor initiates the object and assigns values

        durability--;
        return durability
    }

Methods modify things in the class. There are 2 kinds of methods
  • void
  • returned

    • Voids don't return any variable. Returned methods are equal to the designated variable.
